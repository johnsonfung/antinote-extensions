# Totoro

A light theme for Antinote.

## Installation

1. Download `totoro.json`
2. Open Antinote
3. Go to **File > Open Themes Folder** (or press `Cmd+Shift+T`)
4. Copy the JSON file to the Themes folder
5. Go to **Preferences > Visual & Themes**
6. Select **Totoro** from the theme dropdown

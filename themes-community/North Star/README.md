# North Star ⭐

Welcome to **North Star**! This theme was heavily inspired by Samurize's ⁠Aurora Flare Theme and JFung's Gundam Wing. Made to work at 0% opacity - Specifically in light mode. Pair with Aurora Flare as the dark mode theme for a cleaner experience, navigating lines of code or notes in a cool, immersive digital world. Hope you enjoy it!

# Clean Line Extension

Clean whitespace from the current line.

## Commands

### remove_all_whitespace()
Remove all whitespace from the current line.

**Examples:**
```
remove_all_whitespace()            // Remove all whitespace from current line
```

**Input:**
```
  Hello   World
```

**Output:**
```
HelloWorld
```

### trim_whitespace()
Remove leading and trailing whitespace from the current line.

**Examples:**
```
trim_whitespace()                  // Trim whitespace from current line
```

**Input:**
```
  Hello   World
```

**Output:**
```
Hello   World
```

## Category
Text Formatting

## Data Scope
Line (operates on current line only)

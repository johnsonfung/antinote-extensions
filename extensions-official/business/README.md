# Business Extension

Business metrics and analytics for growth, A/B testing, forecasting, SaaS metrics, and more.

## Commands

### growth()
Analyze percentage growth between consecutive numbers.

Calculates growth rate between consecutive values in your text, useful for analyzing trends over time.

**Examples:**
```
growth()
```

**Input:**
```
100
120
150
```

**Output:**
Shows percentage growth between each consecutive pair (20%, 25%, etc.)

### forecast()
Forecast future values using linear regression and variance analysis.

Provides statistical forecasting based on historical data with confidence intervals.

**Examples:**
```
forecast()
```

## Calculator Templates

The following commands insert interactive calculator templates:

### cac()
Customer Acquisition Cost calculator template.

Calculate how much it costs to acquire each new customer.

### ab_test()
A/B test statistical significance calculator template.

Determine if your A/B test results are statistically significant.

### sample_size()
Sample size calculator for A/B tests template.

Calculate the required sample size for your A/B test to detect a meaningful difference.

### retention_churn()
Retention and churn analysis template.

Analyze customer retention rates and churn patterns.

### ltv()
Customer Lifetime Value calculator template.

Calculate the total revenue expected from a customer over their lifetime.

### arpu()
Average Revenue Per User calculator template.

Calculate average revenue across all users.

### arppu()
Average Revenue Per Paying User calculator template.

Calculate average revenue from paying customers only.

### runway()
Cash runway calculator for startups template.

Calculate how long your startup can operate with current cash reserves.

### saas_pricing()
SaaS pricing strategy helper template.

Plan and analyze different pricing tiers and strategies.
